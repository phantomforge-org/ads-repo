# Phantom Forge Coming Soon Page

A sleek, modern coming soon landing page for Phantom Forge, featuring a countdown timer and email signup form.

## Description

This project is a static HTML/CSS/JavaScript coming soon page that showcases Phantom Forge's upcoming digital interaction platform. It includes:

- Animated countdown timer to launch date
- Email signup form for early access
- Modern glassmorphism design
- Responsive layout for mobile and desktop
- Social media links placeholder

## Features

- **Countdown Timer**: Displays days, hours, minutes, and seconds until launch
- **Email Signup**: Collects emails for early access notifications
- **Responsive Design**: Works on all device sizes
- **Modern UI**: Glassmorphism effects with gradient backgrounds
- **Accessibility**: Proper semantic HTML and form handling

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Flexbox)
- JavaScript (ES6+)
- Google Fonts (Inter)

## How to Use

1. Clone the repository
2. Open `index.html` in your web browser
3. The page will display the countdown and allow email signups

## Customization

- Edit the launch date in `script.js` (currently set to 30 days from page load)
- Modify colors and styling in `style.css`
- Update content in `index.html`
